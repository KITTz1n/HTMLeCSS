Paragrafo e quebras de linha

Voce pode escrever
um paragrafo de qualquer
jeito. basta colocar tudo
no meio do par de tags:
<p> e </p>

Se precisar quebrar uma linha de um paragrafo especifico, so usar a tag <br>

Simbolos especias:

Marca registrada - ®
Copyright - ©
Trade Mark - ™
Euro - €
Libra - £
Yen - ¥
Delta - Δ
Seta para esquerda - ←
Seta para cima - ↑
Seta para direita - →
Seta para baixo - ↓
